# GIt Command Note

    ## ローカルリポジトリの作成
        Git init
    
    ## リモートリポジトリから取得
        Git Clone <Git@Paht>
    
    ## 現在状況の確認
        Git status
    
    ## 差分の確認
        Git diff

    ## ステージングエリアへの登録
        Git Add <FileName>

        ### 全て追加
            --all

    ## コミット
        GIt Commit 

        ### コメント付き
            -m　<Comment>
    
    ## ブランチ確認
        Git branch

        ### ブランチ切り替え
            checkout <Name>
    
        ### 現在のブランチと指定ブランチの統合
            merge <Name>
